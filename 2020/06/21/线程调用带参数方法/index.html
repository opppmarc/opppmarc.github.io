通过Thread发起线程时怎么调用带参数的方法

在.NET Framework中可以通过2种方法实现线程调用带参数的方法:Thread.Start(object),封装线程的方法和参数

第一种：通过 Thread.Start(object) 方法

Start(object) 指导致操作系统将当前实例的状态更改为 Running，并选择提供包含线程执行的方法要使用的数据的对象。 

public void Start (object parameter);

Object Data type 保存引用对象的地址， 可以将任何引用类型（字符串、数组、类或接口）分配给 Object 变量， Object变量还可以引用任意值类型的数据（数值、、 Boolean Char 、 Date 、结构或枚举）。 Object数据类型可以指向任何数据类型的数据，包括应用程序识别的任何对象实例。 在 Object 编译时不知道变量可能指向的数据类型时，请使用。

在线程(函数)的参数位置定义一个object数据类型参数，在运行线程时通过调用Thread类方法Start(object)来传递参数。 

class Program
{
    static void DownloadFile(object filename)
    {
        Console.WriteLine("开始下载");
        Thread.Sleep(1000);     //让线程休眠1000ms
        Console.WriteLine("下载完成");
    }

            static void Main(string[] args)
    {
        Thread t = new Thread(DownloadFile);    //初始化Therad类的新实例 Downloadfile
        t.Start("xxx种子");                     //让t线程开始运行   传递object类参数filename string
        //t.Start('y');                         //让t线程开始运行   传递object类参数filename char
        //t.Start(666);                         //让t线程开始运行   传递object类参数filename int
        Console.WriteLine("Main");
        Console.ReadKey();
    }
}

第二种：封装线程的方法和参数

单独创建一个Thread类，将线程执行的方法和参数都封装到一个类里面，通过实例化该类，方法就可以调用属性来实现间接的类型安全地传递参数。 

class MyThread
{
    private string filename;
    private string filepath;

    public MyThread(string fileName, string filePath)
    {
        this.filename = fileName;
        this.filepath = filePath;
    }

    public void DownFile()
    {
        Console.WriteLine("开始下载" + filepath + filename);
        Thread.Sleep(1000);
        Console.WriteLine("下载完成");
    }
}

class Program
{
    static void Main(string[] args)
    {
        //通过单独创建一个Thread类来传递参数
        MyThread my = new MyThread("xxx.bt", "http://xxxx.com");
        Thread t = new Thread(my.DownFile);     //new 一个Thread对象，传递my的DowanFile方法
        t.Start();
        Console.ReadKey();
    }
}
