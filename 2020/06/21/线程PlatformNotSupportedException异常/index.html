线程PlatformNotSupportedException异常

2020-06-21
C# / Bug
字数统计: 693   |   阅读时长≈ 3 分钟

System.PlatformNotSupportedException:”Operation is not supported on this platform”

详细信息：
System.PlatformNotSupportedException
HResult=0x80131539
Message=Operation is not supported on this platform.
Source=System.Private.CoreLib
StackTrace:
at System.Func`3.BeginInvoke(T1 arg1, T2 arg2, AsyncCallback callback, Object object)
at _010_线程.Program.Main(String[] args) in D:\VSProject\CSharpStudy\Level3\CSStudy3\010-线程\Program.cs:line 13

问题

在vs2019创建.NET Core(3.1.301)的控制台应用程序，执行以下代码： 

class Program
{
    static void Main(string[] args)
    {
        ////Action a = Test;
        Func<int, string, int> a = Test;        //通过委托开起一个线程    
        IAsyncResult ar = a.BeginInvoke(100, "siki", null, null);      //run报错：平台不支持该操作

        Console.WriteLine("Main");
        while (ar.IsCompleted == false)     //如果当前线程没有执行完毕
        {
            Console.WriteLine('.');
            Thread.Sleep(100);      //让Main线程休眠100ms
        }
        int res = a.EndInvoke(ar);      //取得异步线程a的返回值ar
        Console.WriteLine(res);
        Console.ReadKey();
    }

    static int Test(int i, string str)
    {
        Console.WriteLine("test" + i + str);
        Thread.Sleep(100);          //让当前线程休眠100ms
        return 100;
    }
}

运行前代码中没有出现编译错误，但是在运行后在“IAsyncResult ar = a.BeginInvoke(100, “siki”, null, null);”处会返回一个异常：System.PlatformNotSupportedException 。

上网找了下相关异常的处理方式，最后参考了阿水zev的一篇博客。 (传送门)

阿水例子中使用APM的线程代码运行后会出现同样的错误：

原因

异步编程模型(APM)(使用IAsyncResult和BeginInvoke)不再是异步调用的优选方法。从.NET Framework 4.5开始，基于任务的异步模式(TAP)是推荐的异步模型。因此，而且由于异步委托的实现取决于远程处理但.NET Core不存在的功能，BeginInvoke和EndInvoke委托调用不.NET Core支持。（GitHub问题 dotnet/corefx＃5940 中对此进行了讨论）

解决办法

后来直接就在阿水的例子上实验Task.Run了，自己的代码越改越多bug···
改为基于任务的异步模式Task.Run

public delegate string AsyncMethodCaller(int callDuration, out int threadId);
class Program
{
static void Main(string[] args)
{
//基于任务的异步模式(TAP)
AsyncMethodCaller caller = new AsyncMethodCaller(TestMethodAsync);
int threadid = 0;
//开启异步操作 IAsyncResult BeginInvoke
var workTask = Task.Run(() => caller.Invoke(3000, out threadid));
for (int i = 0; i < 10; i++)
{
Console.WriteLine(“其他业务” + i.ToString());
}
//调用EndInvoke，等待异步执行完成
Console.WriteLine(“等待异步方法TestMethodAsync执行完成”);
string res = workTask.Result;
Console.WriteLine(“Compeleted!”);
Console.WriteLine(res);
Console.ReadKey();
}

    //基于任务的异步模式(TAP)
    static string TestMethodAsync(int callDuration, out int threadId)
    {
        Stopwatch sw = new Stopwatch(); //创建一个Stopwatch对象 用来测量运行时间
        sw.Start();     //开始测量运行时间
        Console.WriteLine("异步TestMethodAsync开始");
        for (int i = 0; i < 5; i++)
        {
            //模拟耗时操作
            Thread.Sleep(callDuration);
            Console.WriteLine("TestMethodAsync:" + i.ToString());
        }
        sw.Stop();      //停止策略运行时间
        threadId = Thread.CurrentThread.ManagedThreadId;    //获取当前线程，获取该线程是否属于托管线程池的值
        return string.Format("耗时{0}ms.", sw.ElapsedMilliseconds.ToString());
    }
}
